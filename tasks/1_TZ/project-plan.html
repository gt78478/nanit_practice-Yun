03 · Дизайн и UX
Визуальная концепция для косметического магазина
🎨 Цветовая палитра
  • Основа: пыльная роза + крем-беж
  • Акцент: глубокий бургунди / nude
  • Нейтраль: офф-вайт + тёмный уголь
  • CSS Custom Properties для тем
  • Тёмный режим (опционально)
✍️ Типографика
  • Display: Playfair Display (элегантность)
  • Body: DM Sans / Plus Jakarta Sans
  • Размер: 14–16px base, 32–56px hero
  • Межстрочный: 1.5–1.7 для читаемости
  • Google Fonts — бесплатно
🖼 Компоненты UI
  • Карточки товаров с hover-эффектом
  • Слайдеры (Swiper.js / CSS-only)
  • Toast-уведомления (добавлено в корзину)
  • Скелетон-загрузка для каталога
  • Sticky header при скролле
📱 Адаптивность
  • Mobile-first подход
  • Брейкпоинты: 480 / 768 / 1024 / 1280px
  • Hamburger-меню на мобильных
  • Touch-события для слайдеров
  • Тест в Chrome DevTools
✨ Анимации & детали
  • CSS transitions: 200–300ms ease
  • Hover: scale(1.02) на карточках
  • Fade-in при подгрузке каталога
  • Анимация добавления в корзину
  • Smooth scroll между секциями
04 · Платёжная система
Интеграция оплаты — чилийский рынок
Сервис Приоритет Почему подходит для Чили Сложность интеграции Учебный проект
Mercado Pago ⭐ Первый Доминирующая платформа Латинской Америки, поддержка CLP, карты/трансфер/наличные Средняя Sandbox-режим ✓ Python SDK ✓
Stripe Резерв Международные карты, работает в Чили с 2022, отличная документация Низкая Test mode ✓ Отличная документация ✓
PayPal Опционально Международная аудитория, доверие пользователей Средняя Sandbox ✓ Webhook тесты ✓
Webpay (Transbank) Локально Главная чилийская система, обязательна для местного бизнеса Выше средней SDK Python ✓ но сложнее тестировать
05 · Локализация
Язык и региональные настройки

🌐 Языки интерфейса

Основной язык — испанский (Чили, es-CL). Это включает особенности чилийского испанского: слово «plata» вместо «dinero», обращение «usted» в официальном контексте.

Дополнительный — английский (en) для международной аудитории.

Реализация: JSON-файлы переводов + JS-функция t('key'). Переключатель языка в хедере.

🇨🇱 Региональные форматы

Валюта: CLP (Peso Chileno) — без дробной части, формат: $12.990

Дата: DD/MM/YYYY (чилийский стандарт)

Телефон: +56 (XX) XXXX-XXXX

Доставка: регионы Чили (Santiago, Valparaíso, Biobío и др.)

RUT: поле для налогового номера в форме покупки
