# Технарское задание проекта

## Основные положжения

Создать ИС для сайта онлайн-магазина косметики.

Сайт должен включать пользовательскую часть и админку.

### Пользовательская часть

Регистрация пользователей.

Система корзины и избранных товаров.

Возможность заказа товаров.

### Интеграции

#### Платёжный сервис

Stripe 

Mercado Pago

#### Сервис доставки

Starken

### Прочее

Фронтенд (html+css+js), бэкэнд (fastapi), БД (Postgre) - на отдельных серверах
